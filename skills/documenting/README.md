# Documenting Skill

SSOT documentation system for projects.

## Purpose

Maintain the Single Source of Truth (SSOT) documentation system/memories. This skill handles creation, updates, validation, and navigation of project documentation.

## Activation

This skill is activated:
1.  **Explicitly**: When user asks to "document", "update ssot", "create memory".
2.  **Autonomously**: At the end of a successful task/feature implementation, the agent checks if documentation updates are required.
